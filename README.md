# R2lay
> Short for Rust Relay

A simple TCP relay made in Rust.

## Should support

- [ ] Keep server socket open
- [ ] Reconnect to server (if disconnected) on new client connection
- [ ] IPv6 support
- [ ] IPv4/IPv6 tunnel
- [ ] TCP Proxy Protocol
  * [ ] Version 1
  * [ ] Version 2
